🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my details, skills, projects, certifications, internship experience, and professional achievements.

🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Professional hero section with profile photo and contact information
- About Me section highlighting education and career goals
- Skills showcase with modern card-based layout
- Project portfolio with detailed project descriptions
- Certifications section with downloadable certificate PDFs
- Internship experience with certificate verification
- Contact form integration
- Smooth scrolling and interactive animations
- Clean and modern UI/UX design

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Git & GitHub
